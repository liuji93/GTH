Paper : Optimal Projection Guided Transfer Hashing for Image Retrieval 
Author : JI Liu and Lei Zhang
Published in AAAI 2019
Contact : jiliu@cqu.edu.cn

This package contains cleaned up codes for the AAAI paper, including: 
main_demo.m: the demo test codes  
demo.m: function to train the GTH-g and GTH-h models  
folder "Method-GTH-g": original codes for GTH-g  
folder "Method-GTH-h": original codes for GTH-h  
PIE05.mat: source domain on PIE dataset
PIE29.mat: target domain on PIE dataset
